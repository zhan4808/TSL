These utilities let you create and print TSL eeprom binary images. 

While you can use them direct on files, you usually want to use the batch files in `/programming` to access them.

Build on macOS/Linux with `make` (outputs to `software/eeprom-util/bin/`). Build on Windows via `tsl-eeprom-utils.sln`. Copy new binaries into `programming/eeprom-utils-bin` for use with the factory batch files.
